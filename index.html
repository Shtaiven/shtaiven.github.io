ABOUT ME

Hey, I'm Steven.

Seeing something I've created in motion gives me goosebumps, so I strive to find areas in my life where I can make that happen. I've built a sensor station, an RC truck made of junk, LEGO, and Erector sets. I take electronics apart and put them back together. I've written code that was run on a self-driving car. We live in a world full of opportunities to influence the next generation of robotics and automation, and I'm diving in head-first.

I have a Bachelor's of Engineering in Computer Engineering and have completed Udacity's Self-Driving Car Engineer Nanodegree program.

Skills

SKILLS

Software

I write Python scripts, shell scripts, and full desktop applications. I actively strive to write clean, maintainable, version-controlled code.

Firmware

I've written firmware for PIC, ARM microcontrollers using Nordic's API for Bluetooth LE enabled devices, and Arduino-compatible devices. I've worked in C, C++, and assembly.

Electrical Engineering

From CAD to manufacture and soldering, I've designed circuitry and PCBs that have gone into consumer wearables and automated manufacturing systems.

Robotics

I have experience working with electronic RC cars, self-driving cars, and an automated moving ultraviolet light sensor system.

Machine Learning

I have worked with Keras and scikit-learn to create and train classifiers and convolutional neural networks in Python and C++. I have some experience using R for statistical analysis.

Image Processing

I am familiar with using kernels, image pre-processing for machine learning, and OpenCV.
